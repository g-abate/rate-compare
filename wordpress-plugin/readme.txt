Compare short-term rental rates across multiple booking channels including Airbnb, VRBO, and Booking.com.

== Description ==

Rate Compare is a powerful WordPress plugin that helps property managers and hosts compare rental rates across multiple booking channels in real-time. Whether you're managing Airbnb, VRBO, or Booking.com listings, this plugin provides a comprehensive solution for rate optimization and competitive analysis.

= Key Features =

* **Multi-Channel Support**: Compare rates from Airbnb, VRBO, and Booking.com
* **Real-Time Data**: Get up-to-date pricing information with 15-minute caching
* **Easy Integration**: Simple shortcode and widget integration
* **Responsive Design**: Works perfectly on desktop and mobile devices
* **Admin Dashboard**: Comprehensive settings and configuration options
* **Security First**: Built with WordPress security best practices
* **Performance Optimized**: Efficient caching and minimal resource usage

= How It Works =

1. **Configure Channels**: Add your property URLs from supported booking channels
2. **Set API Keys**: Configure API keys for enhanced data access (optional)
3. **Display Comparison**: Use shortcodes or widgets to show rate comparisons
4. **Monitor Performance**: Track savings and optimize your pricing strategy

= Supported Channels =

* **Airbnb**: Direct integration with Airbnb property listings
* **VRBO**: Support for VRBO property URLs
* **Booking.com**: Integration with Booking.com hotel listings

= Requirements =

* WordPress 5.0 or higher
* PHP 7.4 or higher
* Modern web browser with JavaScript enabled

= Installation =

1. Upload the plugin files to the `/wp-content/plugins/rate-compare` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to 'Rate Compare' in the admin menu to configure your settings
4. Add your property URLs and configure API keys
5. Use the `[rate_compare]` shortcode to display rate comparisons

= Frequently Asked Questions =

= Does this plugin work with all booking channels? =

Currently, Rate Compare supports Airbnb, VRBO, and Booking.com. We're constantly working to add support for more channels.

= Do I need API keys to use this plugin? =

API keys are optional but recommended for enhanced data access and accuracy. The plugin can work with basic web scraping, but API keys provide more reliable data.

= Is this plugin secure? =

Yes, Rate Compare follows WordPress security best practices including input sanitization, output escaping, nonce verification, and capability checks.

= Can I customize the appearance? =

Yes, the plugin includes CSS classes that can be customized to match your theme's design.

= Does this plugin affect site performance? =

No, Rate Compare is optimized for performance with efficient caching and minimal resource usage.
